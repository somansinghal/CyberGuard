# ✨ Features

### 🔐 Password Analyzer

✔ Password strength detection  
✔ Entropy calculation  
✔ Security score meter  
✔ Password complexity breakdown  

---

### 🖥 Cyber Terminal Simulation

```
> Initializing CyberGuard scan...
> Loading password database...
> Running brute force simulation...
> Analyzing entropy...
> Security scan complete.
```

---

### 🎲 Password Generator

Generate secure random passwords instantly.

Example:

```
G8#pX2!sM9@fLq7T
```

---

### 🔎 Hash Generator

Generate cryptographic hashes.

Supported:

```
SHA256
```

Example:

```
5e884898da28047151d0e56f8dc6292773603d0d6aabbdd...
```

---

### 📧 Email Breach Checker

Check if an email might appear in known breaches.

*(Demo mode for educational purposes)*
